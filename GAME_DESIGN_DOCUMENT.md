## 1. 现有系统经济审计

### 1.1 每日经济流建模

#### 收入端（日均）

| 来源 | 理论最大值 | 实际均值 | 备注 |
|------|-----------|---------|------|
| 发薪日 | +50 | +50 | 每日固定 |
| 挖矿（5/h, 24h上限）| +120 | +40~60 | 取决于领取频率 |
| 轮盘（6h冷却，均值~55）| +220 | +110~165 | 每日2~3次 |
| 每日任务 | +85 | +85 | 签到10+抽卡20+传说30+连击25 |
| **抽卡返还** | — | **+169.5** | ⚠️ 关键发现 |
| **日均总收入** | — | **~454~570** | — |

#### ⚠️ 关键发现：抽卡是正期望值行为

```
单次抽卡期望返还计算：
  传说(1.5%) × 50 = 0.75
  极高(5.0%) × 20 = 1.00
  高(15.0%) × 20 = 3.00
  中上(25.0%) × 8  = 2.00
  中等(30.0%) × 8  = 2.40
  中下(15.0%) × 8  = 1.20
  低(8.0%)  × 3   = 0.24
  极低(2.0%) × 3  = 0.06
  ─────────────────────
  期望返还 = 10.65 筹码/次
  抽卡成本 = 5 筹码/次
  净收益 = +5.65 筹码/次 ← 玩家越抽越富！
```

**结论**：当前经济系统存在根本性通胀——核心玩法（抽卡）本身就是印钞机。

#### 消耗端（日均）

| 去向 | 日均消耗 | 备注 |
|------|---------|------|
| 抽卡 | 100~200 | 20~40次 × 5筹码 |
| 道具购买 | 0~30 | 偶尔购买加速卡/幸运符 |
| 皮肤购买 | 0（一次性）| 买完即永久，18个皮肤总价约2,700 |

#### 日均净通胀率

```
日均净盈余 = 454~570（收入）- 100~200（抽卡成本）- 0~30（道具）≈ +224~370 筹码/天
```

**玩家在没有任何策略思考的情况下，每天自动积累 200+ 筹码。**

### 1.2 各系统问题诊断

| 系统 | 问题 | 严重度 | 根因 |
|------|------|--------|------|
| 抽卡 | 正期望值，越抽越富 | 🔴 严重 | 奖励过高，成本过低 |
| 连击 | Level 3（3x奖励+100%稀有率）过于慷慨 | 🟡 中等 | 倍率叠加导致高连击=爆发性通胀 |
| 挖矿 | 5/h太低，2小时才够1次抽卡 | 🟡 中等 | 收益感不足，玩家不关心 |
| 轮盘 | 6h冷却 vs 均值55，性价比低 | 🟡 中等 | 等待成本>期望收益感知 |
| 每日任务 | 签到任务无意义（必然完成） | 🟢 轻微 | 缺少渐进式难度 |
| 皮肤 | 价格100~185，梯度仅85 | 🟡 中等 | 缺少"顶级炫耀"定位 |
| 每日挑战 | 代码已有，UI未实现 | 🔴 严重 | 开发资源未到位 |
| 社交 | 无分享/排行机制 | 🟡 中等 | 缺少社交驱动力 |

## 2. 现有系统平衡性调整建议

### 2.1 抽卡系统调整

> **设计原则**：抽卡应该是"微亏但有趣"的行为，收集的乐趣来自运气而非印钞。

#### 调整方案：奖励分层递减 + 成本微调

| 稀有度 | 现有概率 | 调整后概率 | 现有奖励 | 调整后奖励 | 理由 |
|--------|---------|-----------|---------|-----------|------|
| 传说 | 1.5% | 1.2% | 50 | 35 | 降低正EV，传说应有收藏价值而非经济价值 |
| 极高 | 5.0% | 4.0% | 20 | 15 | 收窄高运奖励 |
| 高 | 15.0% | 12.0% | 20 | 12 | 同上 |
| 中上 | 25.0% | 25.0% | 8 | 6 | 微调 |
| 中等 | 30.0% | 32.0% | 8 | 5 | 基础回报降低 |
| 中下 | 15.0% | 16.0% | 8 | 3 | 惩罚感，但不大 |
| 低 | 8.0% | 7.5% | 3 | 1 | 亏损明确 |
| 极低 | 2.0% | 2.3% | 3 | 0 | 纯亏损，赌场感 |

```
调整后期望返还：
0.012×35 + 0.04×15 + 0.12×12 + 0.25×6 + 0.32×5 + 0.16×3 + 0.075×1 + 0.023×0
= 0.42 + 0.60 + 1.44 + 1.50 + 1.60 + 0.48 + 0.075 + 0
= 6.115 筹码/次

抽卡成本 = 5 筹码
净收益 = +1.115 筹码/次 ← 仍为正，但已大幅收敛

进一步方案：将抽卡成本提升至 8 筹码
净收益 = 6.115 - 8 = -1.885 筹码/次 ← 微亏，但传说仍可回本
```

#### 推荐方案：**抽卡成本 7 筹码，奖励按上表调整**

```
净收益 = 6.115 - 7 = -0.885 筹码/次
日均30次抽卡：净消耗 = -26.5 筹码/天
```

**玩家每日需要通过其他系统"赚"回抽卡成本，形成"赚→抽"循环。**

### 2.2 连击系统调整

> **设计原则**：连击应给"爽感"而非"通胀"。

| 连击等级 | 现有倍率 | 调整后倍率 | 现有稀有率加成 | 调整后稀有率加成 | 理由 |
|----------|---------|-----------|--------------|----------------|------|
| Level 1 (≥3次) | 1.5x奖励 | 1.3x奖励 | +25% | +15% | 降低通胀但保留爽感 |
| Level 2 (≥5次) | 2x奖励 | 1.6x奖励 | +50% | +30% | 同上 |
| Level 3 (≥10次) | 3x奖励 | 2x奖励 | +100% | +50% | 大幅收敛，避免爆发性通胀 |

**连击窗口**：5秒 → **3.5秒**（增加操作压力，使高连击更难达成）

### 2.3 挖矿系统调整

| 参数 | 现有值 | 调整后 | 理由 |
|------|--------|--------|------|
| 产出率 | 5/h | **8/h** | 2小时够2次抽卡，感知价值提升 |
| 上限 | 24h/120 | **24h/192** | 满额领取=27次抽卡，有明确目标感 |
| 新增：离线累计提示 | 无 | **上线时推送"你矿里的N筹码已就绪"** | 回归驱动力 |

### 2.4 轮盘系统调整

| 参数 | 现有值 | 调整后 | 理由 |
|------|--------|--------|------|
| 冷却 | 6h | **4h** | 日均3~6次，更频繁=更有存在感 |
| 最低奖励 | 10 | **15** | 提升保底感 |
| 新增：轮盘道具 | 加速卡 | **加速卡+幸运符+赌神券** | 扩大奖池，增加期待感 |

### 2.5 每日任务调整

| 任务 | 现有 | 调整后 | 理由 |
|------|------|--------|------|
| 签到 | +10，打开即可 | ❌ **移除** | 零决策=零乐趣 |
| 抽卡达人 | 抽5次 +20 | 抽15次 +25 | 有明确目标感，需要一定投入 |
| 传说猎手 | 抽到传说 +30 | **保留**，额外+1赌神券 | 传说本身稀有，奖励可保持 |
| 连击大师 | 5连击 +25 | 达成10连击 +35 | 提高门槛，增加挑战性 |
| **新增**：筹码守卫 | — | 单日抽卡净亏损≤10 +20 | 反向目标：学习"适可而止" |
| **新增**：幸运轮手 | — | 轮盘3次 +15 | 引导体验轮盘系统 |

### 2.6 皮肤价格梯度调整

| 档位 | 现有范围 | 调整后范围 | 定位 |
|------|---------|-----------|------|
| 入门款 | 100~120 | 80~120 | 新手可触碰，3~5天存够 |
| 中端款 | 130~155 | 150~250 | 中期目标，1~2周存够 |
| 高端款 | 160~185 | 300~450 | 炫耀性消费，需要刻意存钱 |
| **新增：至尊款** | — | 600~1000 | 顶级社交货币，需要数周积累 |

> 至尊皮肤建议加入专属入场动画、专属连击特效、专属赌桌背景——不仅仅是颜色变化。

## 机制一：赌神段位系统（Casino Rank）

> **设计目的**：解决"缺少长期目标"问题，给玩家持续追求的进度感  
> **玩家体验**：从无名小卒到赌场传奇的成长旅程  
> **优先级**：🔴 **P0** — 没有段位，游戏就没有灵魂

#### 段位等级与晋升

| 段位 | 图标 | 赌绩分要求 | 解锁特权 | 主题台词 |
|------|------|-----------|---------|---------|
| 赌徒 🎲 | 🎲 | 0 | 基础功能 | "新来的？先学会数筹码吧。" |
| 赌客 🃏 | 🃏 | 100 | 解锁轮盘 | "有点意思，但还差得远。" |
| 赌侠 🎭 | 🎭 | 500 | 解锁赌桌对决 | "赌侠？这个称号你配得上吗？" |
| 赌王 👑 | 👑 | 2000 | 解锁黑市拍卖 | "赌王？难得一见的人物。" |
| 赌神 🌟 | 🌟 | 5000 | 全系统解锁+至尊皮肤折扣 | "赌神...你也配？" |

#### 赌绩分获取规则

| 行为 | 赌绩分 | 频率 | 理由 |
|------|--------|------|------|
| 单次抽卡 | +1 | 无限 | 基础积累 |
| 达成连击 Level 2 | +5 | 每次触发 | 鼓励技巧 |
| 达成连击 Level 3 | +15 | 每次触发 | 高难高奖 |
| 完成每日任务 | +10 | 每日 | 稳定来源 |
| 抽到传说卡 | +25 | 每次触发 | 运气奖励 |
| 赌桌对决胜利 | +8~20 | 每次胜利 | 新机制联动 |
| 完成成就 | +30~100 | 一次性 | 里程碑奖励 |

#### 赌绩分衰减

- **每周一 00:00 衰减 5%**（仅对当前段位内的分数衰减，不降段）
- 例：赌侠段位有 800 分 → 周一变为 760 分
- 一旦达到某段位门槛，**永不降段**
- 衰减目的：鼓励持续活跃，而非一次性冲段

#### 与现有系统联动

| 联动点 | 规则 |
|--------|------|
| 抽卡 | 赌神段位抽卡时，传说概率 +0.5%（从1.2%→1.7%） |
| 连击 | 赌王以上段位，连击窗口 +0.5秒 |
| 皮肤 | 至尊皮肤需赌王以上段位才能购买 |
| 成就 | 新增"赌神降临"成就（达到赌神段位） |

#### 实现复杂度

- 前端：新增段位显示条（类似经验条），新增段位弹窗
- 数据：Supabase 新增 `rank_points`, `rank_level`, `last_decay_date` 字段
- 预估工时：2~3天

## 机制二：牌运占卜（Fortune Divination）

> **设计目的**：每日仪式感 + 核心筹码消耗口 + Pierre Cashon 主题深度契合  
> **玩家体验**：每天登录，先占卜一卦，根据运势决定今天的策略  
> **优先级**：🔴 **P0** — 没有占卜，"牌运预测"游戏名就是假的

#### 核心规则

**每日流程**：
1. 玩家打开占卜台，支付 **15 筹码**
2. 从 22 张塔罗牌中随机抽取 3 张（过去·现在·未来）
3. 3 张牌的组合决定今日运势效果，持续到次日 00:00 重置
4. 每日最多占卜 **3 次**（取最佳结果覆盖），每次 **15 筹码**

#### 经济影响分析

```
每日占卜消耗：15 × 1~3次 = 15~45 筹码/天
日均抽卡30次 × 7筹码 × 10%优惠（愚者效应约20%触发率）≈ 节省 4.2 筹码
净消耗 = 15~45 - 4.2 ≈ 10~41 筹码/天
```

**这是每日必经的筹码消耗口，同时创造了策略思考空间。**

## 机制三：赌桌对决（Table Showdown）

> **设计目的**：赋予收集卡牌实际用途 + 新增策略深度 + 大额筹码消耗口  
> **玩家体验**：从"被运气玩"变成"用手里的牌去博弈"  
> **优先级**：🟡 **P1** — 丰富玩法深度的关键机制

#### 核心规则

**基本流程**：
1. 玩家从已收集的卡牌中选择 5 张组成"手牌"
2. 支付入场费，挑战 AI 庄家
3. 比大小：手牌总战力 vs 庄家手牌总战力 + 随机波动
4. 赢了拿奖金，输了入场费归零

**卡牌战力计算**：

| 稀有度 | 基础战力 | 数量权重 | 理由 |
|--------|---------|---------|------|
| 传说 | 10 | ×2（上限1张）| 传说牌是王牌 |
| 极高 | 7 | ×2（上限2张）| 高价值 |
| 高 | 5 | ×3（上限3张）| 中坚力量 |
| 中上 | 3 | ×5（不限）| 可靠选择 |
| 中等 | 2 | ×5（不限）| 基础填充 |
| 中下 | 1 | ×5（不限）| 炮灰 |
| 低 | 0 | ×5（不限）| 无战力 |
| 极低 | -1 | ×5（不限）| 负面效果 |

**关键设计**：玩家需要"用掉"抽到的卡牌来组牌，同一张牌同时只能出现在一手牌中。

#### 对决难度与奖励

| 难度 | 庄家战力 | 入场费 | 胜利奖金 | 胜率(估算) | 期望值 |
|------|---------|--------|---------|-----------|--------|
| 新手局 | 8~12 | 10 | 18 | ~70% | +2.6 |
| 普通局 | 15~20 | 25 | 50 | ~50% | +0 |
| 高手局 | 22~28 | 50 | 110 | ~35% | -11.5 |
| 赌神局 | 30~40 | 100 | 280 | ~20% | -44 |

**核心洞察**：新手局微正EV引导体验，高难度局负EV消耗筹码。玩家会自然向"自己的水平"靠拢。

#### 随机波动机制

```
最终战力 = 手牌战力 + random(-3, +3)
庄家战力 = 底数 + random(-2, +2)
```

- 随机波动确保"必胜"不存在，但有概率优势
- 高战力手牌赢面大但不是100%

#### 特殊技能牌

| 技能 | 条件 | 效果 | 冷却 |
|------|------|------|------|
| 皮尔卡松的凝视 | 手牌含传说卡 | 对方战力 -3 | 3局 |
| 千王出千 | 手牌含3张以上极高/高 | 我方战力 +2 | 5局 |
| 破釜沉舟 | 筹码 < 入场费×3 | 胜利奖金 ×1.5 | 1局 |

#### 与现有系统联动

| 联动点 | 规则 |
|--------|------|
| 抽卡 | 抽到的卡进入牌库，可用于组牌 |
| 连击 | 对决中达成连胜3局，激活"连胜之火"（+2战力） |
| 占卜 | 审判牌 = 下次对决必胜1次 |
| 段位 | 赌侠以上段位解锁赌桌对决 |
| 成就 | "千王之王"成就（对决累计胜利50次） |

## 机制四：赌场投资（Casino Investment）

> **设计目的**：**核心筹码消耗口** + 风险/回报博弈 + 每日登录驱动力  
> **玩家体验**："把筹码投出去，明天看看是亏是赚"——比挖矿刺激100倍的放置玩法  
> **优先级**：🟡 **P1** — 解决通胀问题的关键机制

#### 三种投资方案

| 投资类型 | 最低投入 | 锁定时长 | 收益规则 | 期望回报 | 风险定位 |
|----------|---------|---------|---------|---------|---------|
| 🏦 安全债券 | 50 | 12h | 确定返还110% | +5 (10%) | 保本微利 |
| 📈 风投基金 | 100 | 24h | 50%概率返还60%，50%概率返还180% | +20 (20%) | 高波动 |
| 🎰 豪赌合约 | 200 | 24h | 20%归零，30%返50%，30%返200%，20%返400% | +60 (30%) | 极高波动 |

#### 详细收益分布

**🏦 安全债券**：
```
投入100 → 12h后确定获得 110
净收益：+10（确定性）
年化收益率（按游戏时间）：~73%
适合人群：保守型玩家、新手
```

**📈 风投基金**：
```
投入100 → 24h后：
  50%概率：收回 60（亏40）
  50%概率：收回 180（赚80）
  期望值：0.5×60 + 0.5×180 = 120，净收益 +20
适合人群：有一定筹码余量，愿意承担风险的玩家
```

**🎰 豪赌合约**：
```
投入200 → 24h后：
  20%概率：归零（亏200）← 这才是真正的赌！
  30%概率：收回 100（亏100）
  30%概率：收回 400（赚200）
  20%概率：收回 800（赚600）← 一夜暴富的幻想
  期望值：0.2×0 + 0.3×100 + 0.3×400 + 0.2×800 = 310，净收益 +110
适合人群：赌徒型玩家、追求刺激的老玩家
```

#### 限制规则

1. **同时只能持有 1 笔投资**（防止分散投资消除风险）
2. **投资期间筹码被锁定，不可撤回**（真正的沉没成本）
3. **豪赌合约需赌客以上段位**（新手保护）
4. **每日最多投资 1 次**（防止无限刷）

#### 经济影响建模

```
假设活跃玩家日均投资分配：
  40% 选择安全债券（均值投入100）→ 日均消耗 -10（净亏，因为110%返还 < 通胀率）
  35% 选择风投基金（均值投入150）→ 日均消耗 -30~+30（波动）
  25% 选择豪赌合约（均值投入300）→ 日均消耗 -60~+90（大波动）

全服日均筹码净消耗 ≈ 5~15 筹码/玩家/天
```

**关键洞察**：虽然期望值为正，但**投资锁定期间筹码不可用**，实际上减少了流通筹码量。加上豪赌的20%归零概率，长期来看是筹码回收机制。

#### 与现有系统联动

| 联动点 | 规则 |
|--------|------|
| 占卜 | 太阳牌效果 = 投资收益 +10% |
| 段位 | 赌王段位 = 安全债券返还115% |
| 成就 | "华尔街赌徒"成就（累计投资10次） |
| 轮盘 | 轮盘可掉落"投资优惠券"（手续费-5%） |

## 机制五：赌侠录（Gambler's Chronicle）

> **设计目的**：社交分享 + 自我表达 + 有机传播  
> **玩家体验**："看看我今天抽到了什么！"→ 分享 → 朋友也想玩  
> **优先级**：🟢 **P2** — 增长型功能，核心闭环已建立后再加

#### 核心功能

##### 1. 个人赌侠卡

一键生成的分享图卡，包含：
- 当前段位 + 赌绩分
- 最稀有卡牌展示
- 今日运势
- 总抽卡次数 / 连击最高记录
- 个人二维码（扫码直接开始游戏）

视觉风格：黑金配色 + 赌场筹码质感 + 皮尔卡松水印

##### 2. 每周排行榜

| 榜单 | 统计周期 | 排名依据 | 奖励 |
|------|---------|---------|------|
| 🔥 运气王 | 每周 | 传说卡抽取次数 | 50筹码+限定称号 |
| ⚡ 连击王 | 每周 | 最高连击记录 | 30筹码+限定称号 |
| 📈 投资王 | 每周 | 投资净收益最高 | 40筹码+限定称号 |
| 🎯 占卜王 | 每周 | 占卜次数最多 | 20筹码 |

- 排行榜数据存 Supabase，每周一 00:00 重置
- 前端展示 Top 20，不显示具体数值（只显示排名和昵称）
- **不实时刷新**，每日更新一次（避免作弊感知）

##### 3. 高光时刻推送

系统自动记录并提示"精彩瞬间"：
- 首次抽到传说 → "值得铭记的时刻！要不要分享？"
- 10连击达成 → "赌神附体！"
- 豪赌合约400%回报 → "一夜暴富！"
- 占卜出"天选之人"组合 → "今日天命所归！"

#### 与现有系统联动

| 联动点 | 规则 |
|--------|------|
| 抽卡 | 传说卡自动触发分享提示 |
| 连击 | 高连击自动触发分享提示 |
| 投资 | 高回报自动触发分享提示 |
| 成就 | 新增"名扬四海"成就（分享5次） |

#### 技术方案

- 赌侠卡：html2canvas 生成图片，前端完成
- 排行榜：Supabase `leaderboard` 表，每日 cron 更新（Supabase Edge Function）
- 分享：Web Share API（移动端）+ 下载图片（桌面端）
