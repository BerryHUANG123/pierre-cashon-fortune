## 🎬 关于这个项目

本游戏以周星驰 1991 年电影 **《赌侠 Ⅱ 之上海滩赌圣》** 中的传奇人物 **皮尔·卡松（Pierre Cashon）** 为人设，打造的一款纯前端牌运预测小游戏。

皮尔卡松——赌坛上那个永远西装革履、眼神深邃的神秘赌神——将亲自为你占卜今日的牌桌运势。是大吉还是大凶？是连胜还是止步？一切尽在他的掌握之中。

## ✨ 游戏功能

### 🃏 牌运预测
- **8 种运势结果**：从「终极王牌」到「大凶」，涵盖所有运势档次
- 每次抽卡皆有动效加持，配以神秘的黑金风格 UI
- 每局记录历史，追踪你的运势走向

### 💰 筹码系统
- 初始筹码，抽卡消耗，高运奖励
- 筹码实时动画显示：涨了发绿光，跌了闪红光
- 筹码归零时触发「破产」保护机制

### 🎯 连击系统
- 连续获得高运可触发「连击」加成，筹码倍增
- 三档连击等级，配有独立动效与浮字特效
- 最高连击记录实时追踪

### 🎡 幸运转盘
- 每日可免费使用的幸运转盘，随机获取筹码奖励
- 转盘旋转动效顺滑，结果反馈即时

### 🏆 成就系统
6 枚可解锁成就，每枚均附皮尔卡松专属台词：

| 成就 | 解锁条件 | 卡松语录 |
|------|----------|----------|
| 初出茅庐 | 首次抽卡 | *"哼，新手？在赌桌上，每个人都是从零开始！"* |
| 小有斩获 | 累计获得 100 筹码 | *"100 筹码？这才哪到哪！不过算是个开始。"* |
| 传说猎手 | 抽到终极王牌 | *"抽到王牌？运气不错！但别高兴太早。"* |
| 皮肤收藏家 | 收集 3 种皮肤 | *"喜欢收集皮肤？品味不错，但还不够！"* |
| 欧皇附体 | 连续 3 次高运 | *"连续高运？看来你今天确实被幸运女神眷顾。"* |
| 连击大师 | 达成 5 连击 | — |

### 📋 每日挑战
- **每日签到**：开局即领筹码
- **速抽挑战**：10 秒内抽卡 3 次
- **精准挑战**：连续 3 次抽中运以上
- **连击挑战**：单次达成 10 连击
